# E-commers-website-Testing
Proud to share my latest project: E-Commerce Test Automation Framework built using Java, Selenium WebDriver, TestNG, Maven, and Page Object Model (POM).
🔹 Project Highlights:
✅ Automated end-to-end user flows for an e-commerce website
✅ Login & Logout validation
✅ Product search and sorting verification
✅ Add to Cart / Remove from Cart scenarios
✅ Checkout flow with validations
✅ Responsive testing across screen resolutions
✅ Cross-browser execution (Chrome & Firefox)
✅ TestNG Suite execution with reusable framework design

🔹 Tech Stack:
💻 Java 17
🌐 Selenium WebDriver
🧪 TestNG
📦 Maven
🏗️ Page Object Model (POM)
⚡ Eclipse IDE

🔹 Key Learnings:
✔ Building scalable automation frameworks
✔ Writing reusable page classes and test methods
✔ Handling waits, assertions, and dynamic elements
✔ Cross-browser automation execution
✔ Real-world QA mindset for regression testing

This project strengthened my hands-on skills in Automation Testing and Java-based test framework development.
